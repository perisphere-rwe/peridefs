# Retrieve the narrative algorithm description for heart failure (v1)

Retrieve the narrative algorithm description for heart failure (v1)

## Usage

``` r
get_hf_v1_defs(variable_type = c("condition", "outcome"))
```

## Arguments

- variable_type:

  `"condition"` (default) or `"outcome"`.

## Value

Character string, or `NULL`.
